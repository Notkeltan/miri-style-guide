02 — Typography

Type System

Two typefaces define MIRI's voice — an elegant serif for authority and presence, paired with a precise sans-serif for clarity and readability.

Headings · Buttons · CTA
IvyJournal
Ivy Journal — serif used for all headings, button labels, and CTA titles. Loaded via Adobe Fonts (Typekit kit czh2yqb). Set as both primary and secondary global typography in Elementor at weight 400.
Use for: H1–H4, CTA text, pull quotes, button labels
Body · UI · Labels
Inter Tight
Inter Tight — variable sans-serif for all body text and UI elements. Loaded self-hosted via Elementor's Google Fonts integration and from the Adobe Typekit kit as "inter-tight-variable". Set as the "text" global typography. Fallback: sans-serif, Font Awesome / eicons for icons.
Use for: Body copy, captions, labels, navigation, metadata

03 — Logo

Logo Variants

MIRI uses two marks: the Singularity logomark (icon only) and a full wordmark lockup. Both are white-on-navy by default. The transparent PNG variants (Bottom Text, Side Text) have no background and sit cleanly on any dark surface. The upscaled mark is navy on transparent — use a CSS filter: brightness(0) invert(1) to render it white on dark backgrounds.

Singularity Mark — White on Navy
Primary usage. Icon-only mark used as favicon, social avatar, and standalone brand element. Always white on dark navy.
Singularity Mark — Dark on Light
Use on light backgrounds (#f0f2f5, white). Invert to dark navy. Never use on coloured backgrounds other than navy or off-white.

Machine Intelligence Research Institute

Full Wordmark — Mark + Name
Used in site navigation and footer. The full organisation name appears in Ivy Journal, letter-spaced. The nav SVG (Group-47.svg) combines both into a single lockup.
Full-Colour Mark — Legacy / Social Avatar Variant
Older full-colour variant with orange, green, and blue feathers and a "MIRI" wordmark. Used as the YouTube channel avatar and in legacy contexts. The current website uses the single-colour white or navy version exclusively. Displayed here on white — the JPG has a white background already so it blends naturally.
